Template Parameters
TThe type of the variable being looked for. If you don't know, use 'object'.
Parameters
objThe starting object to get the value off from.
valueThe value to set into the object.
pathThe series of paths to get to the value. Unity concatonates all of these with a '.'.
Returns
Returns the value found at the path. If something goes wrong, it returns T's default value.
